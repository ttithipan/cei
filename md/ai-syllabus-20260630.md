## Assignment: Blind & Heuristic Searches

- **Group size:** 6–7 students, arranged by yourself
- **Type:** Web application — The Romania-Map Pathfinding problem
  - Implement 1 Blind search and 1 Custom Heuristic search
  - Any arbitrary cities can be selected as start and goal states
  - Compare runtime, space, and path cost between the two searches
- **Grading:** Creativity, Completeness, Presentation skills
