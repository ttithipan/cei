## Topics Overview

### Processes & Protection (Lectures 1–5)
OS fundamentals: what an OS is, protection domains, processes, I/O,
files, and threading primitives.

### Synchronization (Lectures 6–9)
Concurrency, mutual exclusion, atomic instructions, lock implementation,
semaphores, monitors, and the readers/writers problem.

### Scheduling & Deadlock (Lectures 10–13)
Scheduling policies, core concepts, and deadlock detection and prevention.

### Virtual Memory (Lectures 14–16)
Address spaces, paging, page tables, TLBs, and demand paging.

### I/O & File Systems (Lectures 17–20)
General I/O abstractions, disk scheduling, file system design,
and reliability.

### Distributed Systems (Lectures 21–24)
Distributed file systems, networking, data processing, coordination,
and atomic commit protocols.
